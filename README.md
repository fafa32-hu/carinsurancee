# Projects
# [Car Insurance](./CarInsurance)

An MVC web application simulating a car insurance quote system with administrative capabilities.

**Features:**
- User input form for policy information (vehicle details, driver information, coverage options)
- Business logic for calculating insurance quotes based on various risk factors
- Administrative dashboard for viewing all issued quotes
- Data persistence using Entity Framework

**Tech Stack:** ASP.NET MVC, Entity Framework, SQL Server, Razor

# [NFL Stats](./NFLStats)

A data visualization application for storing and displaying football statistics designed for fantasy football analysis.

**Features:**
- MVC architecture with Code First database design
- Entity Framework 6 for data access and management
- Interactive data visualization for football statistics
- CRUD operations for managing player and team data

**Tech Stack:** ASP.NET MVC, Entity Framework 6, SQL Server, Code First workflow

**Status:**  Work in progress

# Getting Started

# Prerequisites
- Visual Studio 2019 or later
- .NET Framework 4.7.2 or later
- SQL Server or SQL Server Express

# Installation

1. Clone the repository:

git clone https://github.com/fafa32-hu/csharp-projects.git

2. Open the solution file (`.sln`) in Visual Studio

3. Restore NuGet packages:
```bash
Update-Package -reinstall
```

4. Update the connection string in `Web.config` to point to your SQL Server instance

5. Run Entity Framework migrations:
```bash
Update-Database
```

6. Build and run the project (F5)
